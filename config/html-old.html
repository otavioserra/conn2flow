PayPal Plus

É necessário ter uma conta no PayPal e um aplicativo do mesmo para poder ativar esta opção. Acesse o seguinte material para poder saber como configurar um aplicativo no PayPal Developers : [link-material-tutorial]

Informações Úteis

  • Para o PayPal Plus funcionar, é necessário informar primeiramente o ClientID e o Secret do aplicativo criado no PayPal Developers aqui no B2make.
  • Com o aplicativo configurado, é necessário escolher a opção ATIVAR para o PayPal Plus funcionar, ou DESATIVAR para o mesmo ficar desabilitado aqui no B2make.
  • Com o aplicativo ativado, é necessário definir a opção SANDBOX para ativar o ambiente de testes de vendas. Ou então a opção LIVE para a loja funcionar em ambiente real.
    • IMPORTANTE: para o correto funcionamente é necessário que esta opção seja também definida no aplicativo do PayPal Developers. Para isso basta acessar o PayPal Developers, acessar o aplicativo e escolher entre a opção SANDBOX / LIVE
