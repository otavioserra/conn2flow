## â• Creating a Prompt

### How to Create
1. Click **"Add Prompt"**
2. Fill in:
   - **Name** - Descriptive name
   - **Category** - Select or create
   - **Prompt Text** - The actual instruction
   - **Variables** - Optional placeholders
3. Click **"Save"**

### Prompt Example
```
Name: Blog Post Intro
Category: Content
Prompt: Write an engaging introduction for a blog post about 
        {{topic}}. Target audience: {{audience}}. Tone: {{tone}}.
        Length: 2-3 paragraphs.
Variables: topic, audience, tone
```

## ğŸ”¤ Using Variables

### What are Variables?
Placeholders you fill in when using the prompt:
- Format: `{{variable_name}}`
- Filled at use time
- Makes prompts flexible

### Example with Variables
```
Prompt: "Summarize the following text about {{subject}} 
         in {{length}} words: {{text}}"

When using:
- subject: "climate change"
- length: "100"
- text: [your text here]
```
