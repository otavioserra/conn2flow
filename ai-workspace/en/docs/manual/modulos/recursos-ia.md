## üéØ AI Targets

### What Are Targets?
Targets define the destination for AI-generated content:
- A specific page
- A component
- A template
- A variable value

### Using Targets
1. Access the **AI Targets** module
2. Create or select a target
3. Specify which resource receives the content
4. Configure field mappings

## üé≠ AI Modes

### What Are Modes?
Modes control the AI's behavior:
- **Tone** - Formal, casual, friendly, professional
- **Length** - Short, medium, detailed
- **Creativity** - Conservative or creative
- **Format** - Plain text, HTML, Markdown

### Common Modes
| Mode | Best For |
|------|----------|
| **Formal Writing** | Business content, proposals |
| **Creative Writing** | Blog posts, marketing |
| **Technical** | Documentation, guides |
| **Casual** | Social media, informal content |

### Selecting a Mode
1. When using AI features, look for mode selector
2. Choose the appropriate mode
3. AI adjusts its output accordingly

## üìù AI Prompts

### What Are Prompts?
Prompts are instructions that tell the AI what to create:
- "Write a product description for..."
- "Create a welcome message that..."
- "Generate 5 headlines about..."

### Writing Good Prompts
```
‚úÖ Good: "Write a 100-word product description for a 
    wireless headphone. Highlight comfort, battery life, 
    and sound quality. Use casual, friendly tone."

‚ùå Bad: "Write about headphones"
```

### Prompt Tips
1. **Be specific** - Include details
2. **Set length** - Specify word count
3. **Define tone** - How should it sound?
4. **Give context** - Who is the audience?
5. **Include examples** - If you have preferences

## üíª Using AI Generation

### Basic Workflow
1. Navigate to a page or component editor
2. Look for the **AI** or **Generate** button
3. Select or enter a prompt
4. Choose a mode (if available)
5. Click **Generate**
6. Review the output
7. Edit if needed
8. Insert into your content

### Editing AI Output
Always review and edit AI content:
- Check for accuracy
- Verify facts and claims
- Adjust tone to match your brand
- Fix any errors

## ‚ùì Frequently Asked Questions

### Q: Why isn't AI generating content?
**A:** Check:
1. Is the AI feature configured? (API keys)
2. Is your prompt clear enough?
3. Are there usage limits?
4. Check error messages

### Q: The content doesn't match what I wanted
**A:** Try:
1. Be more specific in your prompt
2. Try a different mode
3. Provide examples
4. Regenerate with adjusted instructions

### Q: Can AI write entire pages?
**A:** Yes, but we recommend:
1. Generate sections, not entire pages
2. Always review and edit
3. Add your personal touch
4. Verify all information

### Q: Is AI content SEO-friendly?
**A:** AI can write SEO-focused content if you:
1. Include keywords in your prompt
2. Ask for specific formatting (headings, etc.)
3. Request meta descriptions
4. Still optimize manually

### Q: Who owns AI-generated content?
**A:** Content generated for your site is yours to use. However, check your AI provider's terms of service.

## üí° Best Practices

### Content Quality
1. **Always review** - AI makes mistakes
2. **Fact-check** - Verify claims and data
3. **Personalize** - Add your unique voice
4. **Edit** - Refine the output
5. **Don't over-rely** - AI is a tool, not a replacement

### Efficiency
1. **Save good prompts** - Reuse what works
2. **Use modes** - Match tone to content type
3. **Iterate** - Regenerate if needed
4. **Template prompts** - Create templates for common tasks

### Ethical Use
1. **Disclose** - Consider disclosure if appropriate
2. **Original content** - Don't pass off as 100% human-written if not
3. **Avoid misinformation** - Verify before publishing
4. **Respect copyright** - Don't ask AI to copy others' work
