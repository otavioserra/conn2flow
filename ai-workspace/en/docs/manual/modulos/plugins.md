## What are Plugins?

**Plugins** extend Conn2Flow's functionality without modifying the core system. Think of them as add-ons that give your site new capabilities - like contact forms, e-commerce, analytics, or integrations with other services.

## üìã Plugin List

### Understanding the List
Each plugin shows:
- **Name** - Plugin identifier
- **Description** - What it does
- **Version** - Installed version
- **Status** - Active, Inactive, or Needs Update
- **Actions** - Activate, deactivate, configure, delete

### Plugin States
| State | Meaning |
|-------|---------|
| **Active** | Plugin is running and functional |
| **Inactive** | Installed but not running |
| **Needs Update** | Newer version available |
| **Error** | Problem detected |

## ‚ûï Installing a Plugin

### From the Marketplace
1. Click **"Add New"** or **"Browse Plugins"**
2. Search or browse available plugins
3. Click **"Install"** on your chosen plugin
4. Wait for installation to complete
5. Click **"Activate"** to enable it

### Manual Installation
1. Download the plugin package (.zip)
2. Click **"Upload Plugin"**
3. Select the downloaded file
4. Click **"Install Now"**
5. Activate after installation

## ‚è∏Ô∏è Activating & Deactivating

### Deactivating a Plugin
1. Find the plugin in the list
2. Click **"Deactivate"**
3. Plugin stops running but remains installed
4. Settings are preserved

### When to Deactivate
- Troubleshooting issues
- Not currently needed
- Testing if plugin causes problems
- Before major updates

### Reactivating
1. Find the inactive plugin
2. Click **"Activate"**
3. Plugin resumes operation

## ‚ùì Frequently Asked Questions

### Q: Plugin doesn't appear after installation
**A:** Try:
1. Refresh the page
2. Check if it needs activation
3. Verify installation completed successfully
4. Check error logs

### Q: Plugin causing problems
**A:** Steps to diagnose:
1. Deactivate the plugin
2. Check if problem resolves
3. Check for updates
4. Contact plugin support

### Q: Can I use any WordPress plugin?
**A:** No, Conn2Flow has its own plugin format. WordPress plugins are not compatible.

### Q: How many plugins can I install?
**A:** No strict limit, but too many plugins can slow down your site. Only install what you need.

### Q: Plugin conflicts
**A:** If two plugins conflict:
1. Deactivate one to identify the conflict
2. Check for updated versions
3. Contact plugin developers
4. Consider alternative plugins

### Security
1. **Trusted sources** - Only official marketplace
2. **Review permissions** - What access does it need?
3. **Remove unused** - Inactive plugins can be risks
4. **Stay informed** - Follow security announcements
