## What is Categories Management?

The **Categories** module (Admin Categories) helps you organize your content into logical groups. Categories make it easier for users to find related content and help you structure your site's information architecture.

### How to Create
1. Click **"Add Category"**

   - **Name** - Display name (e.g., "Technology")

## â“ FAQ

### Q: Can I have the same category name twice?
**A:** Yes, if they have different parents. Slugs must be unique.

### Q: What happens if I rename a category?
**A:** The display name changes but content associations remain.

### Q: Can I merge categories?
**A:** Not directly. Reassign content to one category, then delete the other.

### Q: How many levels of nesting?
**A:** Technically unlimited, but 2-3 levels is recommended for usability.
