## What is the File Manager?

The **File Manager** (Admin Files) is your central hub for managing all uploaded files, images, documents, and media in your Conn2Flow system. Upload, organize, delete, and find files all in one place.

---

## üéØ Getting Started

### Accessing File Manager
1. On the Dashboard, find the **Admin Files** card
2. Click to open the module
3. You'll see your file library

---

## üìã File Library

### What You'll See
- **Thumbnails** - Visual previews of images
- **File names** - Original or renamed files
- **File types** - Images, documents, PDFs, etc.
- **File size** - How large each file is
- **Upload date** - When it was uploaded
- **Actions** - View, copy URL, delete

### Filtering Files
- **Search** - Find by filename
- **Type filter** - Show only images, documents, etc.
- **Date filter** - Show files from specific periods
- **Folder filter** - Navigate folders

## ‚¨ÜÔ∏è Uploading Files

### How to Upload
1. Click **"Upload Files"**
2. Either:
   - Drag and drop files onto the upload area
   - Click to browse and select files
3. Wait for upload to complete
4. Files appear in your library

### Upload Tips
- Maximum file size depends on server settings
- Supported formats: Images, PDFs, documents
- Compress large images before uploading
- Use descriptive filenames

---

## üìÅ Organizing Files

### Creating Folders
1. Click **"New Folder"**
2. Enter folder name
3. Click **"Create"**

### Moving Files
1. Select files to move
2. Click **"Move"**
3. Choose destination folder
4. Confirm

### Renaming Files
1. Click on file
2. Click **"Rename"**
3. Enter new name
4. Save

---

## üîó Using Files

### Getting File URL
1. Click on a file
2. Click **"Copy URL"**
3. URL is copied to clipboard
4. Paste anywhere you need it

### Inserting in Content
- Use the media button in page editor
- Browse and select from library
- Or paste the copied URL

## üóëÔ∏è Deleting Files

### Single File
1. Click on file
2. Click **"Delete"** (trash icon)
3. Confirm deletion

### Multiple Files
1. Select multiple files
2. Click **"Bulk Actions"**
3. Select **"Delete"**
4. Confirm

## ‚ùì FAQ

### Q: What's the maximum file size?
**A:** This depends on server configuration. Typically 2MB-50MB.

### Q: Can I upload videos?
**A:** Check your server settings. For large videos, consider using YouTube/Vimeo and embedding.

### Q: A file is missing
**A:** Check:
1. Did the upload complete successfully?
2. Are you in the right folder?
3. Try the search function.

### Q: Can I organize files into folders?
**A:** Yes! Create folders to organize your files by project, date, or type.

## üí° Best Practices

1. **Name files clearly** - "hero-banner-2024.jpg" not "img123.jpg"
2. **Organize into folders** - Keep things tidy
3. **Optimize images** - Compress before uploading
4. **Clean up regularly** - Delete unused files
5. **Backup important files** - Keep local copies
