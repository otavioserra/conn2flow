# üìÑ Pages Management - User Manual

## What is Pages Management?

The **Pages** module (also called Publisher Pages) is where you create and manage all the content pages of your website. Think of it as your content editor where you write, format, and publish articles, blog posts, or any other page content.

---

## üéØ Getting Started

### Accessing Pages
1. From the Dashboard, find the **Publisher - Pages** card
2. Click to open the module
3. You'll see a list of all your pages

---

## üìã Page List

### Understanding the List
Each page shows:
- **Title** - The page headline
- **Status** - Draft, Published, or Scheduled
- **Author** - Who created it
- **Date** - When it was created/published
- **Actions** - Edit, preview, delete buttons

### Filtering Pages
- **Search** - Find pages by title or content
- **Status filter** - Show only drafts, published, or scheduled
- **Category filter** - Show pages in specific categories
- **Date filter** - Show pages from specific periods

---

## ‚ûï Creating a New Page

### Step by Step
1. Click **"Add Page"**
2. Fill in the basic fields:
   - **Title** - Your page headline (required)
   - **Content** - The main text of your page
3. Add optional elements:
   - Featured image
   - Categories
   - Tags
   - SEO settings
4. Choose to **Save Draft** or **Publish**

---

## ‚úçÔ∏è The Content Editor

|--------|----------|
| **B** | Bold text |
| **I** | Italic text |

| **Image** | Insert an image |
| **List** | Bulleted or numbered list |
| **Heading** | H1, H2, H3 headings |

### Tips for Good Content
1. Use headings to organize content
2. Keep paragraphs short (3-4 sentences)
3. Add images to break up text
4. Use bullet points for lists
5. Include a clear call-to-action

## üì∏ Featured Image

### Adding a Featured Image
1. Find the **Featured Image** section (usually right sidebar)
2. Click **"Set Featured Image"**
3. Upload a new image or select from media library
4. Click **"Select"**

### Image Guidelines
- Recommended size: 1200x630 pixels
- Formats: JPG, PNG
- Keep file size under 500KB for faster loading

## üè∑Ô∏è Categories and Tags

### Categories
Categories are broad groupings:
- Check one or more categories
- Click "Add New Category" to create more
- Example: "News", "Products", "Blog"

### Tags
Tags are specific keywords:
- Type tags separated by commas
- Example: "marketing, social media, tips"
- Help with search and related content

---

## üîç SEO Settings

### Optimizing for Search Engines
1. **Meta Title** - The title shown in search results
   - Keep under 60 characters
   - Include keywords
2. **Meta Description** - The preview text in search results
   - Keep between 150-160 characters
   - Make it compelling to click
3. **URL Slug** - The page address
   - Use dashes between words
   - Keep it short and descriptive

---

## üìÖ Publishing Options

### Save Draft
- Page is saved but not visible to visitors
- You can continue editing later
- Use for work-in-progress

### Publish
- Page immediately goes live
- Visitors can see it on the website
- You can still edit after publishing

### Schedule
1. Click on the date/time selector
2. Choose when to publish
3. Save - page will auto-publish at that time

### Unpublish
- Takes the page offline
- Content is preserved
- Can be republished anytime

## ‚ùì Frequently Asked Questions

### Q: My page isn't showing on the website
**A:** Check:
1. Is the status "Published"?
2. Is the publish date in the past?
3. Is the page linked in the menu?

### Q: Can I duplicate a page?
**A:** Look for a "Duplicate" or "Clone" option in the actions menu.

### Q: How do I add a video?
**A:** Paste the video URL (YouTube, Vimeo) in the editor - it will auto-embed.

### Q: Images look stretched
**A:** Use images with correct aspect ratio. Resize before uploading if needed.
