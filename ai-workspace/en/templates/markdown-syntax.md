````markdown
# Markdown Syntax

## Headings
Headings are defined with the '#' symbol followed by a space. The number of '#' indicates the heading level.
# Heading level 1
## Heading level 2
### Heading level 3

## Emphasis
Emphasis can be added using '*' or '_' for italics and '**' or '__' for bold.
*Italic text*
**Bold text**

## Lists
Unordered lists are defined with '-' or '*'.
- Item 1
- Item 2
Ordered lists are defined with numbers followed by a dot.
1. First item
2. Second item

## Links
Links are defined with text in brackets followed by the URL in parentheses.
[Link text](http://example.com)

## Images
Images are defined similarly to links, but with a '!' before.
![Alt text](http://example.com/image.jpg)

## Code Blocks
Code blocks are defined with three backticks (```).
```
Code here
```

## Blockquotes
Blockquotes are defined with '>'.
> This is a blockquote.

## Tables
Tables are defined with '|' and '-' to separate headers and rows.
| Header 1 | Header 2 |
|----------|----------|
| Row 1    | Row 1    |
| Row 2    | Row 2    |

## Comments
Comments are defined with '//' for single line and '/* ... */' for multiple lines.
````
