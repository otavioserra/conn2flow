Parameter Documentation
Basic Properties
name
Descriptive search-form name
id
Unique widget identifier
description
Optional description
GET Action
Full submission uses GET and keeps the term in the search parameter.
form_action
Results page; an empty value falls back to pages-index-search/
Search and Autocomplete
The name="search" input is intrinsic and required.
Autocomplete starts at 3 characters with a 300 ms debounce.
Results use batches of 30, keyboard navigation, and a local cache.
Templates use the input-search and results-box cells.
Extra Fields
fields contains optional additional filters only; never add the search field there.
