JSON de Todos os Parâmetros
Documentação dos Parâmetros
Propriedades Básicas
name
Nome descritivo do formulário de busca
id
Identificador único do widget
description
Descrição opcional
Ação GET
O envio completo usa GET e mantém o termo no parâmetro search.
form_action
Página de resultados; vazio usa pages-index-search/
Busca e Autocomplete
O input name="search" é intrínseco e obrigatório.
O autocomplete inicia com 3 caracteres e debounce de 300 ms.
Resultados são paginados em lotes de 30, com navegação por teclado e cache local.
Templates usam as células input-search e results-box.
Campos Extras
fields contém somente filtros opcionais adicionais; nunca é necessário cadastrar o campo search.
name
Nome técnico do filtro
label
Rótulo visível
type
Tipo do campo
required
Obrigatoriedade do filtro extra
options
Opções ou limites específicos do tipo
