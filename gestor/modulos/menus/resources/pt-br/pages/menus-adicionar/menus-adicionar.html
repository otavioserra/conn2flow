Itens do Menu

Monte a estrutura do menu adicionando itens de diferentes tipos. Arraste pelo para reordenar; arraste para a direita/esquerda para aninhar como filho ou voltar ao nível anterior (estilo WordPress). Clique em um item para selecioná-lo: o próximo item será inserido logo abaixo dele.

Conteúdo do Menu

O HTML abaixo é o template renderizado no site. Use o bloco <!-- item < --> ... <!-- item > --> para itens sem filhos (folhas) e o bloco <!-- item-parent < --> ... <!-- item-parent > --> para itens com submenu — dentro dele, o placeholder [[item#children]] recebe a renderização recursiva dos filhos. Variáveis por item: [[item#label]], [[item#url]], [[item#slug]] e [[item#css_classes]]. Use <!-- no-item < --> ... <!-- no-item > --> para o estado sem itens.
