Site Settings User Settings Email Settings Language Settings PayPal

Site Settings

Configure the basic identity of your site. The site name is used in emails, page titles and system references.

User Settings

Manage user authentication security. Enable reCAPTCHA to protect login and registration forms against bots and automated attacks.

reCAPTCHA V2 (Checkbox)

Email Settings

Configure the SMTP server for sending system emails such as notifications, password resets and contact form responses.

Language Settings

Configure the default language, language selection widget and automatic browser language detection for visitors.

PayPal Integration

Configure your PayPal REST API credentials and integration mode. Use Standard to use credentials from this panel, or Gateway to use credentials managed by the Payment Gateways module.

API Credentials

Enter your PayPal REST API credentials. You can find them at PayPal Developer Dashboard.
