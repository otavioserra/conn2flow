Configurações do Site

Configure a identidade básica do seu site. O nome do site é utilizado em emails, títulos de páginas e referências do sistema.

Configurações de Usuário

Gerencie a segurança de autenticação dos usuários. Ative o reCAPTCHA para proteger formulários de login e cadastro contra bots e ataques automatizados.

Métodos de Login

Para configurar: Acesse o Google Cloud Console, crie um projeto, vá em "Tela de consentimento OAuth", depois em "Credenciais" -> "Criar credenciais" -> "ID do cliente OAuth" (Aplicativo Web). Adicione a URI de redirecionamento abaixo em "URIs de redirecionamento autorizadas".

Cadastre esta URI exatamente assim no Google Cloud Console.

Para configurar: Acesse o portal Meta for Developers, crie um aplicativo do tipo "Consumidor", adicione o produto "Login do Facebook", vá em "Configurações do Login do Facebook" e insira a URI de redirecionamento abaixo em "URIs de redirecionamento do OAuth válidas".

Cadastre esta URI exatamente assim no painel de Apps da Meta.

Configurações de Email

Configure o servidor SMTP para envio de emails do sistema como notificações, redefinição de senhas e respostas de formulários de contato.

Configurações de Linguagem

Configure o idioma padrão, widget de seleção de idioma e detecção automática do idioma do navegador dos visitantes.

Integração PayPal

Configure suas credenciais da API REST do PayPal e o modo de integração. Use Padrão para usar credenciais deste painel, ou Gateway para usar credenciais gerenciadas pelo módulo de Gateways de Pagamento.

Credenciais da API

Insira suas credenciais da API REST do PayPal. Você pode encontrá-las no Painel de Desenvolvedor PayPal.
