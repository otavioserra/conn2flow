Documentação dos Parâmetros
Propriedades Básicas
name - Nome descritivo do formulário
id - Identificador único do formulário
description - Descrição opcional do formulário
Controle de Acesso
access_max_simple - Máximo de envios simples por IP (sem CAPTCHA)
access_max - Máximo total de envios por IP
force_recaptcha - Força uso de reCAPTCHA v3 sempre
Configuração de Email
email.recipients - Destinatários separados por ";"
email.reply_to - Email para resposta
email.reply_to_name - Nome do remetente para resposta
email.subject - Assunto do email (#code# = número único)
email.message_component - ID do componente de template de email
Redirecionamentos
redirects.success.type - Tipo: "url" ou "page"
redirects.success.path - Caminho para redirecionamento de sucesso
redirects.error.type - Tipo de redirecionamento de erro
redirects.error.path - Caminho para redirecionamento de erro
Campos do Formulário
Propriedades dos Campos:
type - Tipo: "text", "email", "textarea", "select", etc.
name - Nome único do campo (usado como chave)
label - Rótulo exibido para o usuário
required - Campo obrigatório (true/false)
placeholder - Texto de dica no campo
validation - Regras de validação adicionais
Funções da Biblioteca
formulario_controlador():
Prepara e configura formulários no frontend
Carrega componentes HTML e JavaScript
Configura validações client-side
Gerencia permissões de acesso
Integra reCAPTCHA v2/v3
Processa schema de campos e redirecionamentos
formulario_processador():
Processa envios de formulários no backend
Valida dados e CAPTCHA
Aplica rate limiting e anti-spam
Salva dados em forms_submissions
Envia emails de notificação
Gerencia redirecionamentos pós-envio
