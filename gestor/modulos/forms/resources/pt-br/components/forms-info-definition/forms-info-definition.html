JSON de Todos os Parâmetros
Documentação dos Parâmetros
Propriedades Básicas
São utilizados para identificar o formulário.
name
Nome descritivo do formulário
id
Identificador único do formulário
description
Descrição opcional do formulário
Campos Nome e Email (Obrigatórios)
Identificador dos campos de nome e email no "fields" para serem utilizados na comunicação por email.
field_name
Identificador do campo de nome no "fields"
field_email
Identificador do campo de email no "fields"
Controle de Acesso (Opcional)
Os controles de acesso são feitos por padrão, mas podem ser configurados conforme necessário por formulário.
access_max_simple
Máximo de envios simples por IP (sem CAPTCHA)
access_max
Máximo total de envios por IP antes de bloqueio por 24 horas
force_recaptcha
Força uso de reCAPTCHA v3 sempre
Configuração de Email (Opcional)
A configuração de email já é feita automaticamente no módulo Ambiente Config / Configurações de Email. Mas pode ser personalizada por formulário.
email.recipients
Destinatários separados por ";"
email.reply_to
Email para resposta
email.subject
Assunto do email
email.template
ID do componente de template de email
Redirecionamentos
URLs de redirecionamento quando houver sucesso num formulário. Se houver fracasso, volte ao mesmo formulário. Ou para raiz do site.
redirects.success.type
Tipo: "url" de acesso que será utilizado para redirecionamento de sucesso
redirects.success.path
Caminho para redirecionamento de sucesso
redirects.error.type
Tipo: "url" de acesso que será utilizado para redirecionamento de erro
redirects.error.path
Caminho para redirecionamento de erro
Campos do Formulário
Pode incluir quantos campos quiser em cada formulário, mas deve seguir a configuração padrão abaixo.
Propriedades dos Campos:
name
Nome do campo (identificador único)
label
Rótulo exibido para o usuário
type
Tipo do campo (text, email, textarea.)
required
Campo obrigatório
Funções da Biblioteca
Sequência da execução das funções na biblioteca formulários para ter uma ideia de como funciona cada configuração acima.
formulario_controlador():
Prepara e configura formulários no frontend
Carrega componentes HTML e JavaScript
Configura validações client-side
Gerencia permissões de acesso
Integra reCAPTCHA v2/v3
Processa schema de campos e opções
formulario_processador():
Processa envios de formulários no backend
Valida dados e CAPTCHA
Aplica rate limiting e anti-spam
Salva dados em forms_submissions
Envia emails de notificação
Gerencia redirecionamentos pós-envio
