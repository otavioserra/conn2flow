Opções

Definição de quais opções um conteúdo / conjunto de conteúdos apresenta nos diversos módulos do sistema responsáveis por manipular conteúdos.
Add-This:
Ferramenta de copartilhamento de conteúdo com outros sistemas como Facebook, Orkut, etc. Para maiores informações acesse: http://www.addthis.com/
Não Indexar:
Selecione para que o conteúdo não seja indexado pelos robots.
Não Mostrar em Buscas:
Selecione para que o conteúdo não seja mostrado nos processos de buscas dentro do sistema.
Página Sem Layout Principal:
Selecione para que o conteúdo não tenha layout da página principal. Útil para quando tem a necessidade de criar uma landing page.
Menu Principal:
Selecione para que o conteúdo faça parte do menu principal do portal. Útil para quando tem a necessidade de se criar um menu principal com sub-itens.
Identificador Auxiliar:
Selecione para criar um Identificador Auxiliar único. Útil para ser usado por ferramentas que necessitem que o conteúdo tenha um identificador fixo que podem mudar o Título e em consequência o Identificador Principal e mesmo assim continuem a identificar algum elemento, como por exemplo um conteiner.
Conteiner Posição: Efeito: #conteiner_posicao_efeito# Tempo:  
Selecione para que o conteúdo tenha um conteiner de posicionamento em um conteiner específico. Defina o efeito de transição ( demo dos efeitos ) bem como o tempo em milisegundos que durará a transição. Defina a posição x e y padrão aqui e se quiser definir uma posição específica para um conteúdo dentro de um conjunto de conteúdos é só definir dentro do conteúdo, que o mesmo irá sobrescrever esse padrão. Essa opão é para uma navegação animada entre os conteúdos como se os mesmos fizessem parte de um mapa.
Comentários:
Selecione para vincular ao(s) conteúdo(s) comentários publicados por usuários comuns do sistema em relação ao(s) mesmo(s) conteúdo(s). Não é necessário ser usuário registrado no sistema para enviar comentráios, apenas preencher um pequeno formulário de indentificação.

Configuração

Definição de quais campos do conteúdo / conjunto de conteúdos serão usados em um conteúdo qualquer. Ou seja, como será o conteúdo em relação aos seus tipos de informações diferentes como um título, um texto, uma imagem, etc. Cada um desses campos serão usados no layout do conteúdo em alguma posição pré-fixada no layout.
Título:
Título é o título do conteúdo, tanto pode ser usado para definição do título do layout do conteúdo, como para ser o nome desse conteúdo e seu identificador único dentro dos conteúdos. Esse campo será usado por motores de busca de informação tais como google, yahoo, bing, etc.
Valor no Layout: titulo
Título Imagem:
Título imagem é uma imagem qualquer que serve para ser usado como imagem de título do layout do conteúdo ao invés de usar o título textual. Útil quando se quer definir uma imagem como título ao invés de um texto.
Valor no Layout: titulo_img
Sub-Título:
Sub-título é usado para ser o sub-título do layout do conteúdo. Útil quando se está diagramando textos que tenham sub-títulos.
Valor no Layout: sub_titulo
Autor:
Autor é usado para definir um autor para um conteúdo.
Valor no Layout: author
Data:
Data de criação ou modificação desse conteúdo ou ainda da ocorrência futura do conteúdo.
Valor no Layout: data
Texto:
Defina o texto do conteúdo, ou seja, qualquer informação textual. No uso do editor HTML (TinyMCE) é possível colocar no corpo do conteúdo uma série de informação. Ou seja, tudo que o editor permitir de inserção de informação.
Valor no Layout: texto
Texto 2:
Defina o texto 2 do conteúdo, ou seja, qualquer informação textual. No uso do editor HTML (TinyMCE) é possível colocar no corpo do conteúdo uma série de informação. Ou seja, tudo que o editor permitir de inserção de informação.
Valor no Layout: texto2
Link Externo:
Link Externo é um endereço de internet ao qual o conteúdo está vinculado com outra informação externa ao conteúdo.
Valor no Layout: link_externo
Imagem Pequena:
Imagem pequena é um campo que permite o envio de uma imagem pequena.
Valor no Layout: imagem_pequena
Imagem Grande:
Imagem grande é um campo que permite o envio de uma imagem grande.
Valor no Layout: imagem_grande
Música:
Música é um campo que permite o envio de uma música mp3.
Valor no Layout: musica
Galeria de Imagens:
Galeria de Imagens é um campo de vinculação de uma galeria de imagens com o conteúdo. Útil para ser usado quando um conteúdo contêm uma série de imagens.
Valor no Layout: galeria
Galeria de Imagens Grupo:
Galeria de Imagens Grupo é um campo de vinculação de um grupo de galeria de imagens com o conteúdo. Útil para ser usado quando um conteúdo contêm uma série de galerias de imagens específica.
Valor no Layout: galeria_grupo
Galerias de Imagens Todas:
Galerias de Imagens Todas é uma vinculação de todas as galerias de imagem com o conteúdo. Útil para ser usado na publicação de todas as galerias de imagens.
Valor no Layout: galeria_todas
Galeria de Vídeos:
Galeria de Vídeos é um campo de vinculação de videos com o conteúdo. Útil para ser usado quando um conteúdo contêm uma série de vídeos.
Valor no Layout: videos
Galeria de Vídeos Grupo:
Galeria de Vídeos Grupo é um campo de vinculação de um grupo de videos com o conteúdo. Útil para ser usado quando um conteúdo contêm uma série de galerias de vídeos específicos.
Valor no Layout: videos_grupo
Galerias de Vídeos Todas:
Galerias de Vídeos Todas é uma vinculação de todos os vídeos com o conteúdo. Útil para ser usado na publicação de todos as galerias de videos.
Valor no Layout: videos_todas
E-Commerce Serviço:
E-Commerce Serviço é uma vinculação de um serviço vendável com o conteúdo.
Valor no Layout: servico
E-Commerce Produto:
E-Commerce Produto é uma vinculação de um produto vendável com o conteúdo.
Valor no Layout: produto
Comentários Facebook:
Comentários Facebook é uma vinculação de um widget de comentários com o conteúdo.
Valor no Layout: comentarios_facebook
Menu Navegação:
Menu Navegação é uma vinculação de um menu gerado automaticamente do local onde o conteúdo se encontra em relação aos demais conteúdos do portal.
Valor no Layout: menu_navegacao
Conteúdos Relacionados:
Conteúdos Relacionados é uma vinculação de uma lista de conteúdos filhos de um conteúdo que é do tipo Grupo de Conteúdos e que tenha conteúdos filhos desse conteúdo em específico. Ou seja, o conteúdo que tiver filhos terá no corpo do conteúdo em um local específico do conteúdo um menu de acesso dos conteúdos filhos desse conteúdo. Útil para ser usado quando se quer mostrar o relacionamento entre os conteúdos, por exemplo um conteúdo produtos que tenha uma série de produtos será mostrado um menu de acesso aos seus produtos.
Valor no Layout: conteudos_relacionados
Parâmetros:
Parâmetros é um campo de vinculação de uma série de parâmetros com o conteúdo. Útil para ser usado quando um conteúdo contêm uma série de campos que não são definidos por padrão.
Valor no Layout: parametros
Categoria:
Selecione para que o conteúdo seja do tipo categoria. Se for uma categoria de produtos nessa posição aparecerão os produtos dessa categoria.
Valor no Layout: categoria

Layout

Definição da estrutura visual do layout do conteúdo / conjunto de conteúdos. Com essa opção é possível definir posicionamento e regras HTML dos campos acima escolhidos.
Usar Personalizado:
Selecione essa opção para ativar o uso do layout personalizado. Se essa opção não estiver selecionada o sistema irá usar um layout definido previamente, ou seja, um layout padrão dos conteúdos.
Layout:
Defina ou modifique o layout a seguir. A regra para um campo ser posicionado no layout é o seguinte: Nos campos acima veja o Valor no Layout e coloque '#valor#'. Por exemplo para o campo Título coloque o valor #titulo# no layout. Use '<!-- valor < -->#valor# com elementos HTML<!-- valor > -->' caso for desabilitar algum campo no futuro, senão use apenas '#valor#'. Sempre mantenha #info_acima# e #info_abaixo# definidos em algum lugar no layout pois o sistema se utiliza desses valores para colocar mais informações, como por exemplo o AddThis.

Imagens

Definição dos padrões relacionadas com os título imagens, imagens grandes e imagens pequenas do conteúdo / conjunto de conteúdos.
Recorte Título Imagem:
 
Resolução máxima que os Título Imagens podem ter para não serem recortadas nessa resolução. Se a imagem for menor que essa resolução mantém a resolução original. O recorte é feito no centro das imagens nessa resolução definida.
Recorte Imagem Pequena:
 
Resolução máxima que as Imagens Pequenas podem ter para não serem recortadas nessa resolução. Se a imagem for menor que essa resolução mantém a resolução original. O recorte é feito no centro das imagens nessa resolução definida.
Recorte Imagem Grande:
 
Resolução máxima que as Imagens Grandes podem ter para não serem recortadas nessa resolução. Se a imagem for menor que essa resolução mantém a resolução original. O recorte é feito no centro das imagens nessa resolução definida.
