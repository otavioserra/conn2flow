# req-064: Atualização de Versões dos Componentes do GitHub Actions (Fim de Vida do Node 20)

Este lote de trabalho (BATCH-064) visa eliminar os avisos de depreciação do Node.js 20 reportados pelos runners do GitHub Actions durante a execução dos pipelines de release. A atualização para versões mais recentes das ações oficiais do GitHub garante compatibilidade com o runtime Node.js 24 padrão.

---

## 1. Modificações nos Workflows do GitHub Actions

Devem ser atualizadas as versões das seguintes ações nos arquivos de workflow do repositório:

### 1.1 Atualizações no Workflow de Release do Gestor
No arquivo [.github/workflows/release-gestor.yml](file:///c:/Users/otavi/OneDrive/Documentos/GIT/conn2flow/.github/workflows/release-gestor.yml):
- Atualizar todas as ocorrências de `actions/checkout@v4` para `actions/checkout@v7`.
- Atualizar todas as ocorrências de `actions/setup-node@v4` para `actions/setup-node@v6`.
- Atualizar a ocorrência de `softprops/action-gh-release@v2` para `softprops/action-gh-release@v3`.

### 1.2 Atualizações no Workflow de Release do Instalador
No arquivo [.github/workflows/release-instalador.yml](file:///c:/Users/otavi/OneDrive/Documentos/GIT/conn2flow/.github/workflows/release-instalador.yml):
- Atualizar a ocorrência de `actions/checkout@v4` para `actions/checkout@v7`.
- Atualizar a ocorrência de `softprops/action-gh-release@v2` para `softprops/action-gh-release@v3`.
