# req-050: Escape HTML, Correção de Abas Aninhadas e Novos Modelos no Módulo Formulários (Forms)

Este lote de trabalho (BATCH-050) define correções no comportamento do editor HTML e abas do módulo Forms, e adiciona 4 novos templates de formulários estilizados via Tailwind CSS (totalizando 8 arquivos traduzidos).

## 1. Modificações no html-editor e forms (Escape HTML)

### 1.1 Modificar `gestor/bibliotecas/html-editor.php`
- Localizar a assinatura da função `html_editor_componente($params = false)`.
- Adicionar suporte ao parâmetro `'html_specialchars'` (booleano) via `$params`.
- Se `html_specialchars` for `true`, realizar o escape dos placeholders `#pagina-html#` e `#pagina-html-extra-head#` diretamente no `$html_editor` antes de retorná-lo.
- **Implementação recomendada**:
```php
	if (isset($html_specialchars) && $html_specialchars) {
		$html_val = isset($html) ? $html : '';
		$html_extra_head_val = isset($html_extra_head) ? $html_extra_head : '';

		$html_editor = modelo_var_troca_tudo($html_editor, '#pagina-html#', htmlspecialchars($html_val, ENT_QUOTES, 'UTF-8'));
		$html_editor = modelo_var_troca_tudo($html_editor, '#pagina-html-extra-head#', htmlspecialchars($html_extra_head_val, ENT_QUOTES, 'UTF-8'));
	}
```

### 1.2 Modificar `gestor/modulos/forms/forms.php`
- Localizar a função `forms_prepare_editor_page`.
- Atualizar a chamada `html_editor_componente($params)` para incluir `'html_specialchars' => true` e passar `$html` e `$html_extra_head` no array `$params`:
```php
	$params = [
		'modulo' => $modulo,
		'alvo' => 'forms',
		'alvos_modelos' => 'forms',
		'target_variables' => forms_item_variables(),
		'html_specialchars' => true,
		'html' => $html,
		'html_extra_head' => $html_extra_head,
	];
	$params[$modo] = true;
```

## 2. Correção de Colisão nas Abas Aninhadas (Tabs Bug)

### 2.1 Modificar `gestor/modulos/forms/forms.js`
- Isolar o contexto de inicialização dos menus de abas Fomantic UI na tela de adição/edição para evitar que a aba principal perca o foco e suma ao alternar sub-abas.
- Alterar as configurações de tab na inicialização (linhas ~18 e ~25):
```javascript
    $('.menuForms .item').tab({
        context: 'parent',
        onLoad: function (tabPath) {
            localStorage.setItem(gestor.moduloId + 'formsActiveTab', tabPath);
            if (tabPath === 'forms-template') schedulePreview(true, true);
        }
    });

    $('.menuFormsTemplate .item').tab({
        context: '[data-tab="forms-template"]',
        onLoad: function (tabPath) {
            if (tabPath === 'forms-preview') schedulePreview(true, true);
            if (tabPath === 'forms-editor' && typeof window.contentPageTabHandler === 'function') window.contentPageTabHandler();
            if (tabPath === 'forms-widget') updateWidgetCodeTab();
        }
    });
```
